# 0006. Umbrella brand deferred; `yokel` is a codename

- Status: Accepted (decision parked, intentionally)
- Date: 2026-06-14

## Context

`yokel` was chosen on a whim — the persona of a local yokel trying to give back. It has personality
and is the owner's, but as a public brand for an open civic project it may not translate to outside
contributors (e.g. SF Civic Tech) and is mildly self-deprecating. Product names (DocketClock,
Watershed Watch) are already settled and strong.

## Decision

Keep `yokel` as the repo **codename** for now. Defer the public umbrella-brand decision until the
project actually opens to contributors / goes public. A GitHub repo rename is cheap (auto-redirects),
so this is low-stakes and reversible.

Candidate directions if/when we rename (recorded so the thinking isn't lost):

- **Watch/signal motif** (ties the product family together): Lookout, Belfry/Townbell, Beacon.
- **Voice / being heard** (the monitor→act→impact loop): Heard, Holler, Aye.
- **Civic-commons gravitas**: Commons, Neighbor.

## Consequences

- No momentum lost to bikeshedding now; the name travels with the repo and can change later.
- Revisit this ADR before any public launch or contributor onboarding.
